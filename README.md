⚔️ War Sim: Strateji Oyunu
Bu proje, Kocaeli Üniversitesi Bilgisayar Mühendisliği programı kapsamında geliştirilmiş, 2D grafiklere sahip bir strateji simülasyon oyunudur.

🚀 Temel Özellikler
Gelişmiş Harita Sistemi: Dinamik olarak oluşturulan ve engeller barındıran savaş alanı.

Birim Çeşitliliği: Farklı yeteneklere ve saldırı gücüne sahip asker sınıfları.

Grafik Motoru: SDL2 kütüphanesi kullanılarak geliştirilen akıcı görsel deneyim.

🛠️ Kullanılan Teknolojiler
Dil: C

Kütüphaneler: SDL2, SDL2_image, SDL2_ttf

Dosya Yönetimi: JSON tabanlı veri saklama
